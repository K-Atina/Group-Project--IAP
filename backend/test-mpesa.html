M-Pesa Payment Test

Test the M-Pesa STK Push integration

Format: 254 followed by 9 digits (no spaces)

API Endpoints

  • POST /api/mpesa/initiate - Initiate STK Push
  • POST /api/mpesa/callback - M-Pesa callback (used by Safaricom)
  • POST /api/mpesa/query - Query payment status

Request Format (initiate):

{
  "phone": "254712345678",
  "amount": 100,
  "event_id": 1
}

Success Response:

{
  "success": true,
  "message": "STK Push initiated successfully",
  "checkoutRequestId": "ws_CO_123456789",
  "order_id": 1
}

Error Response:

{
  "success": false,
  "message": "Error description"
}
